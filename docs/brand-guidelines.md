# CHEEK Brand Guidelines

## 1. Brand Essence
- **Core Identity:** A queer-focused fashion brand balancing "sexy" and "silly", "cheeky" and "chic".
- **Voice & Tone:** Confident, playful, inclusive, and intentionally non-serious. 
- **Mission:** To provide high-quality, sustainable, and conceptual garments for queer individuals who want to play with their gender and express themselves.

## 2. Colour Palette
| Role | Hex | Usage |
| **Primary** | `#df4d7b` | Buttons, Active States, Key Call-to-Actions |
| **Secondary** | `#f6bb93` | Secondary buttons, accent icons |
| **Tertiary** | `#ffc7db` | Tertiary buttons, accent icons |
| **Accent** | `#5bba89` | Decorative elements, hover states |
| **Background** | `#f0e4c9` | Main page background |
| **Text/Anchor** | `#1a1a1a` | Body text, headers, border lines |

## 3. Typography
| Type | Font Name | Usage |
| :--- | :--- | :--- |
| **Heading** | Nerko One | Page Titles, Hero Headlines, Logos |
| **Body** | Outfit | Paragraphs, Buttons, UI Labels, Nav |

## 4. Logo Usage
*All logo files are located in `/public/images/`.*
- **Primary:** Full colour logo for white/cream backgrounds.
- **Monochrome:** Black (`#1a1a1a`) or White versions for use over images.
- **Clear Space:** Maintain a buffer zone around the logo equal to 20% of its total width.
